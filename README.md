This is the public issue tracker for RackMind. The app source code is in a private repository — this repo exists so you can report bugs, request features, and track progress in the open.

## How to contribute

- **Found a bug?** [Open a bug report](https://github.com/rackmind-ai/feedback/issues/new?template=bug_report.yml) with steps to reproduce.
- **Have an idea?** [Submit a feature request](https://github.com/rackmind-ai/feedback/issues/new?template=feature_request.yml) and describe what you'd like to see.
- **Quick question or general chat?** Join the [Discord](https://discord.gg/WHU4VdgWsd).

## What is RackMind?

RackMind is a desktop app that lets you manage your homelab infrastructure through natural language. Connect your Proxmox server, Linux box, Docker host, or Synology NAS — then chat with an AI agent that can create containers, install software, monitor resources, and troubleshoot issues.

Download it at [rackmind.ai](https://rackmind.ai).
